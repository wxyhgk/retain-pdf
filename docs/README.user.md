# 普通用户说明

## 这是什么

这是一个可以把英文 PDF 翻译成中文、同时尽量保留原始排版的网页工具。  
它不是简单把整篇文章拆成纯文本再翻译，而是会尽量保留：

- 原来的段落位置
- 双栏排版
- 公式
- 图片位置
- 整体阅读顺序

如果你平时看英文论文、技术文档、报告材料，这个工具会比普通复制粘贴式翻译更接近“直接得到一份中文版 PDF”。

## 它能帮你做什么

- 把英文科研论文翻译成中文
- 尽量保留原 PDF 的版式
- 保留大部分公式和图文结构
- 生成可以直接阅读、分享和存档的译文 PDF

简单理解就是：

`上传 PDF -> 自动处理 -> 下载中文版 PDF`

## 和普通翻译工具有什么区别

普通翻译工具通常只能做两件事：

- 把一段文字翻译出来
- 或者把整页内容变成纯文本

这样做的问题是，原来的排版、双栏结构、公式位置、图文关系很容易丢掉。  
而这个工具更强调的是“保留阅读体验”。

也就是说，翻译完成后，你拿到的不是一堆散掉的中文文字，而是一份尽量保持原始版式的中文 PDF。

## 适合哪些人

- 需要读英文论文的学生
- 做科研的老师和研究人员
- 需要快速看英文技术资料的人
- 想把英文 PDF 变成更容易阅读的中文版本的人

## 目前最适合的文档

当前最适合的是：

- 英文科研论文
- 技术类 PDF
- 公式较多的文档
- 双栏排版论文

## 暂时不建议的文档

目前效果可能不够稳定的情况包括：

- 极其复杂的海报式排版
- 纯扫描且识别质量很差的 PDF
- 图片里文字非常多的材料
- 特别花哨的杂志式设计稿

## 网页怎么用

使用流程很简单：

1. 打开网页
2. 上传 PDF
3. 等待系统处理
4. 查看任务状态
5. 下载结果

如果文档页数不多，通常很快就能看到结果。  
如果文档较长，系统会先完成解析，再进行翻译和重建排版。

## 目前网页开放的重点能力

当前网页主要面向“学术论文保留排版翻译”。  
也就是说，它最擅长的是论文类英文 PDF，而不是所有类型的文件都一样强。

如果你拿来处理论文，效果通常会更稳定。  
如果你拿来处理非常杂乱的 PDF，结果可能会波动。

## 为什么这个工具值得试

- 不是只翻译文字，而是尽量保留版式
- 比普通复制粘贴式翻译更适合读论文
- 公式、段落、双栏结构会尽量保留
- 结果是 PDF，阅读体验更接近原文

## 需要提前知道的事

- 任何 PDF 翻译都不可能对所有文档 100% 完美
- 原始 PDF 越规整，效果通常越好
- OCR 识别质量会影响最后结果
- 某些特别复杂的页面，仍然可能需要后续继续优化

## 一句话总结

如果你想要的不是“把英文复制出来翻成中文”，而是“尽量保留原排版地读中文版 PDF”，那这个网页就是为这个目标做的。
