# 项目说明

## 项目定位

本项目用于做 PDF 的保留排版翻译。  
目标不是把文档简单转成纯文本，而是在尽量保留原始版式、公式、图片位置、栏布局和阅读顺序的前提下，生成可直接阅读的中文译文 PDF。

## 主要能力

- 支持本地 `json + pdf` 输入
- 支持原始 PDF 直接接入 MinerU 解析
- 支持整本 PDF 翻译
- 支持学术论文场景
- 支持双栏论文
- 支持公式保留
- 支持跳过代码、参考文献、部分元信息
- 支持左右对照 PDF 输出
- 支持命令行、API、前端三种使用方式

## 当前适合的文档类型

- 科研论文
- 技术报告
- 方法说明文档
- 公式较多的英文 PDF

## 当前不强调的方向

- 纯扫描件的极端复杂排版修复
- 通用出版级精修排版
- 通用 Office 文档处理

## 使用方式概览

项目目前有两条主链路：

- 本地 OCR 链路
  已有 OCR 产物时，直接读取 `.json + .pdf` 进行翻译和渲染
- MinerU 链路
  只有原始 PDF 时，先调用 MinerU 获取结构化结果，再翻译和渲染

## 工程特点

- 可接入外部大模型
- 可接入本地部署模型
- 输出结构统一，便于自动化
- 已经具备服务化能力，方便接前端
- 整体工程以稳定可跑为优先

## 当前推荐模式

- 日常使用优先选择 `sci` 模式
- 需要更快时可使用 `fast`
- 需要更细处理时可使用 `precise`
